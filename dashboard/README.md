# Ralpha Dashboard

SaaS-style control UI for the multi-agent system: **Dashboard**, **Agents**, **Runs**, **Settings**, **Logs**.

## Run locally

```bash
cd dashboard
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Stack

- Vite + React + TypeScript + Tailwind CSS  
- Lucide icons  
- Optional API proxy: `/api` → `http://127.0.0.1:8000` (configure a FastAPI backend later)

## Build

```bash
npm run build
npm run preview
```
